# 抖音标注工具

抖音网页版标注工具，支持画笔、橡皮擦、撤销等功能，可通过快捷键快速调用。

## 功能特性

### 核心功能
- ✏️ **画笔工具**：支持红色画笔标注，可通过滚轮调整大小
- 🧽 **橡皮擦**：圆形橡皮擦，可擦除标注，可通过滚轮调整大小
- ↩ **撤销功能**：可撤销上一步操作
- 🗑️ **清屏功能**：一键清除所有标注
- 💾 **保存功能**：将标注内容保存为PNG图片，白色背景
- ⏩ **视频加速**：鼠标左键长按实现视频加速
- ⌨️ **快捷键支持**：多种快捷键组合，避免冲突
- 🎨 **现代化UI**：采用渐变背景、圆角设计
- 🚀 **流畅拖动**：支持自由拖动工具位置
- 🔍 **实时光标反馈**：橡皮擦光标大小随实际大小同步变化

### 快捷键
- `Ctrl+Shift+D`：显示/隐藏工具
- `Alt+Shift+D`：备选快捷键1
- `Ctrl+Alt+D`：备选快捷键2
- `X`：清屏（画布激活时）
- `E`：切换画笔/橡皮擦（画布激活时）
- `S`：保存标注为PNG图片（画布激活时）
- `Ctrl+Z`：撤销操作（画布激活时）
- `Esc`：关闭工具（画布激活时）

## 安装方法

1. 安装 Tampermonkey 浏览器扩展
2. 点击浏览器右上角 Tampermonkey 图标
3. 选择「添加新脚本」
4. 复制 `抖音标注工具.user.js` 的内容
5. 粘贴到 Tampermonkey 编辑器中
6. 点击「文件」→「保存」或按 `Ctrl+S`
7. 刷新抖音网页，工具即可生效

## 使用说明

1. 打开抖音网页版 `https://www.douyin.com/`
2. 按快捷键 `Ctrl+Shift+D` 显示标注工具
3. 点击画笔图标开始标注
4. 悬停主按钮可展开更多功能
5. 拖动主按钮可调整工具位置
6. 再次按快捷键可隐藏工具

## 功能详解

### 画笔模式
- 点击主按钮进入画笔模式
- 鼠标左键拖动即可绘制
- 支持红色线条，默认宽度为2px
- **大小调整**：使用鼠标滚轮可调整画笔大小（范围：1-20px）

### 橡皮擦模式
- 从展开菜单中点击「🧽 橡皮擦」
- 鼠标左键拖动即可擦除标注
- 橡皮擦为圆形，默认直径20px
- **大小调整**：使用鼠标滚轮可调整橡皮擦大小（范围：5-50px）
- **实时光标**：光标大小会随橡皮擦大小同步变化

### 撤销操作
- 从展开菜单中点击「↩ 撤销」
- 可撤销最近的标注操作
- 支持多步撤销，最多保存100步历史记录

### 清屏功能
- 从展开菜单中点击「🗑️ 清屏」
- 一键清除所有标注内容
- 同时清空历史记录

### 视频加速功能
- **开启方式**：开启画布后即可使用
- **使用方法**：鼠标左键长按视频区域
- **功能效果**：长按期间视频自动加速播放
- **停止方式**：松开鼠标左键恢复正常速度

### 保存功能
- **开启方式**：开启画布后即可使用
- **使用方法**：
  - 从展开菜单中点击「💾 保存」按钮
  - 或按快捷键 `S`
- **功能效果**：将当前画布上的标注内容保存为PNG图片
- **图片特性**：
  - 白色背景
  - 包含所有标注内容
  - 文件名自动生成，包含时间戳
  - 格式：`标注截图_时间戳.png`

## 注意事项

1. 该工具仅适用于抖音网页版
2. 确保 Tampermonkey 扩展已启用
3. 如遇到快捷键冲突，可尝试使用其他快捷键组合
4. 工具默认位于右上角（距离右侧15px，顶部25px）
5. 拖动时工具会精确跟随鼠标
6. 修复了位置显示问题，确保工具正确显示在预期位置
7. 简化了拖动实现，提高了拖动流畅度
8. 优化了显示/隐藏动画，避免影响工具位置
9. 视频加速功能需在开启画布后使用
10. 长按鼠标左键时视频会加速播放，松开恢复正常速度
